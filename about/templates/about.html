About This Project

This isn't really much of a web application - it's more like a project containing disparate pluggable Django "apps". This page is actually an app all on its own - it's probably the most portable (or applicable) module of this project as we're probably porting it over to our final project.

We're actually very boring and unimaginative people - hence the lack of content - hell, we couldn't even come up with our own CSS, we borrowed it from the Django Admin pages. If you've got any ideas for project ideas, please don't hesitate to email us at cmpt470-team-awesome@sfu.ca

Anyways, we've got a few apps here. Right now, there's this about page, the admin site, and a polls app, with a few more (hopefully) on the way and a half finished blog.

About This "App"

If you're looking at the source code for this page on our GitHub Repo then you might have noticed that this page (more like template for this page) exists in a different templates directory - the one inside the "about" app, not the one in the project's root directory. This is because of portability reasons. Just in case you want to recycle a particular Django app, i.e. port it to another app, but want to keep the view (template) the same, you can. You can even control the granularity like we did, making this page rely on our project's "base.html".

Cool stuff eh?
